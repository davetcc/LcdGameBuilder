This Library provides an easy way to write basic platform games for 20x4 display.

Installation
--------------------------------------------------------------------------------

To install this library, just place this entire folder as a subfolder in your
Arduino/lib/targets/libraries folder.

When installed, this library should look like:

libraries/LcdGameBuilder                 (this library's folder)
libraries/LcdGameBuilder/LEDDisplay.cpp  (the library implementation file)
libraries/LcdGameBuilder/LEDDisplay.h    (the library description file)
libraries/LcdGameBuilder/keywords.txt    (the syntax coloring file)
libraries/LcdGameBuilder/examples        (the examples in the "open" menu)
libraries/LcdGameBuilder/readme.txt      (this file)
